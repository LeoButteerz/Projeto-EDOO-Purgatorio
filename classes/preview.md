# Pasta de Classes
Todos os arquivos de classes, sejam eles headers ou arquivos para definir métodos (.h; .hpp; .cpp), devem ser colocados dentro dessa pasta.
Caso necessário, mais pastas podem ser colocadas dentro dessa pasta para uma melhor arquitetura.
Atenção para concatenação de pastas, pastas dentro de pastas só devem ocorrer um máximo de 3 vezes.
> /Pasta/Arquivo.h -> RECOMENDADO
> /Pasta/Pasta2/Arquivo.h -> OK
> /Pasta/Pasta2/Pasta3/Arquivo.h -> CUIDADO
> /Pasta/Pasta2/Pasta3/Pasta4/Arquivo.h -> NÃO RECOMENDADO
